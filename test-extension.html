YouTube Tab-Fullscreen Extension Test

Extension Installation Test

To test the extension:

  1. Load the extension in Chrome from the dist/ folder
  2. Go to any YouTube video (e.g., https://www.youtube.com/watch?v=dQw4w9WgXcQ)
  3. Look for the "TF" button in the player controls
  4. Click it or press Alt+T to toggle tab-fullscreen
